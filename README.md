# Студио Присма — Static Demo Site

Демо сайт за **Студио Присма** (prizmastudio.bg) — персонализиран интериорен дизайн.

## 📁 Структура на проекта

```
prizma-studio/
├── index.html        # Основен HTML файл (Single-page)
├── styles.css        # Всички CSS стилове (mobile-first)
├── script.js         # Vanilla JavaScript
├── vercel.json       # Vercel конфигурация
└── README.md
```

## 🌐 Секции на сайта

- **Hero** — пълноекранна снимка с анимирана заглавна секция
- **Статистики** — ключови числа с анимирани брояч
- **Проекти** — галерия с филтриране и lightbox
- **Услуги** — всички 6 услуги с икони и hover ефекти
- **За нас** — профили на двамата архитекти
- **Процес** — 4-стъпков работен процес
- **Отзиви** — слайдер с клиентски мнения
- **Контакти** — форма + контактна информация
- **Footer** — навигация, услуги, социални мрежи

## 🚀 Deploy на Vercel

### Метод 1 — GitHub + Vercel (препоръчителен)

1. Качи файловете в GitHub репозитори
2. Влез в [vercel.com](https://vercel.com)
3. Кликни **"New Project"** → **"Import Git Repository"**
4. Избери репото → Vercel автоматично открива `vercel.json`
5. Кликни **"Deploy"** — готово! 🎉

### Метод 2 — Vercel CLI

```bash
npm install -g vercel
cd prizma-studio/
vercel
```

### Метод 3 — Drag & Drop

Влез в [vercel.com/new](https://vercel.com/new) и преплъзни папката директно.
